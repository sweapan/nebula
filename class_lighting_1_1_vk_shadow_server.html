Lighting::VkShadowServer Class Reference

#include <vkshadowserver.h>

Detailed Description

Vulkan shadow server.

Implements Moment shadow maps for global lights, http://cg.cs.uni-bonn.de/en/publications/paper-details/peters-2015-msm/ Variance shadow maps for local lights (spot, point), http://developer.download.nvidia.com/SDK/10/direct3d/Source/VarianceShadowMapping/Doc/VarianceShadowMapping.pdf
