Game::EntityManager Class Reference

#include <entitymanager.h>

Detailed Description

Keeps track of all existing entites.

Contains state with categories and the world database.

Copyright
(C) 2018-2020 Individual contributors, see AUTHORS file

Generally, you won't need to access any of the methods or variables within this manager directly.

The entity manager handles all entity categories in the game. Categories are collections of properties, arranged as a table where each row is an instance, mapped to an entity.
