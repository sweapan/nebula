#include <animclip.h>

Detailed Description

An animation clip is a collection of related animation curves (for instance all curves required to animate a character).

◆ PrecomputeKeySliceValues()

void CoreAnimation::AnimClip::PrecomputeKeySliceValues ( )

pre-compute the key range values

Precompute the 2 key slice values (first key index and key slice size).

A key slice is the memory range of all curve-keys at a given key index. The numbers must be pre-computed because only non-static curves have keys in the key-slice.
