Win32::Win32Mouse Class Reference

#include <win32mouse.h>

Detailed Description

Overrides the default Mouse input device class and provides raw mouse movement data via DirectInput.

This is necessary because Windows WM_MOUSEMOVE messages stop at the screen border.
