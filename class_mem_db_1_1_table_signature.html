Detailed Description

Basically a bitfield with packed PropertyIds.

Can be used to identify tables, or be used as a mask to query the database for tables that contain a certain set of properties.

Note
This class runs on the assumption that we never create a bitfield that contains more bits than necessary. eg, a field that is 256 bits large but is zero in left most 128 bits is forbidden.
