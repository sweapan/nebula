CoreGraphics::RenderEventHandler Class Reference

#include <rendereventhandler.h>

Detailed Description

A render event handler object is notified by the RenderDevice about noteworthy events.

To react to those events, derive a class from RenderEventHandler, and attach to the render device via RenderDevice::AttachEventHandler().
