Materials Namespace Reference

Detailed Description

Materials represent a set of settings and a correlated type, which tells the engine which shader to use and how.

The material pool provides a chunk allocation source for material types and instances.

A material type declares the draw steps and associated shaders.

The material server is the central hub for the material types.
