It will make sure that no accidential memory leaks are reported by the debug heap.

inlinestatic

register a type (templated)

TYPE must be trivially copyable and destructible, and also standard layout.

Essentially a POD type, but we do allow non-trivially-constructible types since properties are created by copying the default value, not with constructors. The reason for this is because it allows us to do value initialization in declarations.
